Project Overview
--------------------------------------------
This project analyzes a large-scale e-commerce dataset (from Kaggle) to uncover business insights using Python and Excel. It demonstrates professional-grade data cleaning, analysis, visualization, and dashboard reporting. The workflow and deliverables are designed for academic and industry standards.

Key Steps & Methods
--------------------------------------------
1. Data Cleaning:
   - Removed duplicates, handled missing values, type conversions
2. Feature Engineering:
   - Added calculated columns (sales, month, region mapping)
3. Exploratory Analysis:
   - Descriptive stats, distribution analysis, advanced visuals 
4. Visualization:
   - Created charts in Python (matplotlib & seaborn), matched analytics in Excel
5. Reporting:
   - All cleaned/summarized data exported to Excel with dashboards and pivots
6. Documentation:
   - Professional synopsis (Word), executive summary (PowerPoint), structure overview (README)
